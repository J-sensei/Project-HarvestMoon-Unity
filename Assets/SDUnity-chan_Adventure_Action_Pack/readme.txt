Though we try to have maximum compatibility with <SD Unity-Shan 3D Model Data> which provided by Unity Technology Japan in version 1.0, we have more focus on convenience than compatibility. from this version

Therefore, you should ignore readme included in version 1.0.

- This asset is motion package which using <SD Unity-Chan 3D Model  Data> that is provided by Unity Technology Japan. Optimized ‘Unity-Chan Model Data’ has been included.

- Add-bone of every motion has motion value. If you DO NOT use add-bone motion, you should use _Humanoid_only.Mask.


*1.0버전은 ‘유니티 테크놀로지 재팬’에서 제공하는 <SD 유니티짱 3D모델 데이터> 와 호환성을 위주로 제작되었으나 이번 버젼부터는 호환성을 일부 포기하고 모션제작의 편의성을 높였습니다. 때문에 1.0버전의 readme 내용은 지금 버젼에 적용되지 않습니다. 

- 이 어셋은‘유니티 테크놀로지 재팬’에서 공식으로 제공되는, <SD 유니티짱 3D모델 데이터>를 활용한 모션 패키지입니다. 최적화(Optimaze)작업을 한 유니티짱 모델 데이터가 포함되있습니다. 

- 모든 모션의 add-Bone에도 모션값이 있습니다. add-bone 모션을 사용하지 않을 경우엔 _Humanoid_only.Mask를 사용하십시오.
